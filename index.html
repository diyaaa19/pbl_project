Plant Disease Classification

"Automated identification of plant pathogens using transfer learning and deep convolutional neural networks to classify diseases into bacteria, fungi, viruses, oomycetes, and healthy plants."

Problem Statement

Plant diseases cause significant crop losses worldwide, affecting food security and agricultural economies. Early and accurate identification of the causative pathogen is critical for effective treatment. Manual identification requires expertise and is time-consuming. This project develops an automated classification system using deep learning to identify the pathogen type responsible for plant diseases from leaf images.

Literature Review

Existing CNN-based plant disease classification systems focus on disease identification but often don't classify by pathogen type. Transfer learning with models like ResNet, VGG, and EfficientNet has shown promise in agricultural image classification tasks, achieving 85-95% accuracy on various plant disease datasets.

Research Gap / Innovation

Unlike disease-specific classifiers, this project classifies by pathogen category (Bacteria, Fungus, Virus, Oomycete, Nan/Healthy), enabling pathogen-level treatment recommendations. Comparison of ResNet50 vs. EfficientNet architectures provides insights into optimal model selection for agricultural deployments.
